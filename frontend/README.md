# Reactts-Base

## Project Overview

Reactts base is a repo that will act as a base / reference for new porjects that uses React Vite with Typescript

## Requirements

- **Node.js**: v22.7.0
- **Package Manager**: Yarn

## Development Tools

To ensure code consistency and quality, this project uses the following tools:

1. Prettier (Code Formatter)
   Prettier automatically formats your code on save. Install the Prettier extension in your IDE, and enable auto-formatting.

2. ESLint (Linter)
   ESLint is used to catch potential errors and enforce coding standards. Install the ESLint extension in your IDE and enable auto-linting on save.

## Setup Instructions

### 1. Clone the Repository

To begin, clone the repository to your local machine:

git clone <repository-url>
cd aviator-analyzer

### 2. Install Dependencies

Ensure that you have Yarn installed. If not, install it globally:

npm install -g yarn

Then, install the dependencies:

yarn install

### 3. Run the Application

Start the development server:

yarn dev
